# 名词解释

## 格式说明

本文档用于定义业务专业术语及其对应的字段映射，帮助理解业务概念与数据字段的对应关系。

### 文件格式

| 列名 | 说明 | 必填 |
|------|------|------|
| 名词 | 业务术语名称 | 是 |
| 含义 | 业务术语的解释说明 | 是 |
| 对应字段 | 数据字典中对应的字段名称 | 是 |
