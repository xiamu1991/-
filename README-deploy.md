OAH Analysis Dashboard Deployment

This prototype is a static HTML page.

Public URL options

1. GitHub Pages
- Create a new GitHub repository.
- Rename or copy oah-analysis-dashboard.html to index.html.
- Upload index.html to the repository root.
- In GitHub repository Settings -> Pages:
  - Source: Deploy from a branch
  - Branch: main
  - Folder: /root
- Wait 1-2 minutes.
- Your public URL will look like:
  https://<your-account>.github.io/<repo-name>/

2. Netlify
- Open https://app.netlify.com/drop
- Rename or copy oah-analysis-dashboard.html to index.html.
- Drag index.html into the page.
- Netlify will generate a public URL immediately.

3. Vercel
- Create a new project in Vercel.
- Upload index.html.
- Deploy.
- Vercel will generate a public URL.

Recommended local file
- Source file: oah-analysis-dashboard.html
- Publish file name: index.html
