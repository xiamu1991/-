# 指标说明

以下是各个指标的详细说明：

## Discount Amount - Dataset
- **指标别名**：折扣金额，net discount，net discount_amt
- **业务含义**：FP&A 折扣金额
- **口径**：Dataset口径，常用分析口径
- **计算公式**：-
- **涉及字段**：discount_amt
- **样例SQL**：
SELECT 
        SUM(discount_amt) 
FROM 
        sbux_cdm.dwd_pmt_disct_redemption_line_di 
WHERE 
        ds = '20260302' 

## Discount Amount - Accounting
- **指标别名**：Accounting折扣金额，discount_amt，折扣金额
- **业务含义**：Accounting 折扣金额
- **口径**：Accounting结算口径
- **计算公式**：-
- **涉及字段**：discount_amt
- **样例SQL**：
SELECT 
        SUM(discount_amt)/100 
FROM 
        sbux_cdm.dwd_pmt_disct_line_di 
WHERE 
        ds = '20260302' 
        and is_accounting_non_discount = 0

## Discount Amount - OAH
- **指标别名**：折扣金额（排除Srkit分摊）,discount，discount amount，discount_amt，折扣金额
- **业务含义**：折扣金额
- **口径**：OAH口径
- **计算公式**：Discount Amount-Srkit defer
- **涉及字段**：discount_amt,srkit_defer
- **样例SQL**：
SELECT
        SUM(discount_amount) - SUM(Srkit_def)
FROM
        sbux_cdm_dev.dwd_aop_mkt_order_line_discount_detail_di
WHERE
        ds = '20260319'

## O2O Redemption Category Sales
- **指标别名**：Redemption Sales，核销商品金额
- **业务含义**：核销折扣的商品实付金额
- **聚合方式**：⚠️必须先按order_line_no取Max(pay_amt)后再做SUM
- **口径**：OAH口径，O2O报表口径
- **计算公式**：-
- **涉及字段**：pay_amt
- **样例SQL**：
SELECT 
        pos_key
        ,SUM(pay_amt)
FROM 
        (
                SELECT 
                        order_Id
                        ,order_line_no
                        ,pos_key
                        ,MAX(pay_amt) AS pay_amt 
                FROM 
                        sbux_cdm_dev.dwd_aop_mkt_order_line_discount_detail_di 
                WHERE 
                        ds = '20260302' 
                GROUP BY 
                        order_Id
                        ,order_line_no
                        ,pos_key
        ) AS A
GROUP BY 
        pos_key

## O2O Redemption Pay Amount
- **指标别名**：核销预付费券的整单支付金额，pay_amt，pay_amount
- **业务含义**：使用了预付费券的订单对应的整单支付金额
- **聚合方式**：⚠️必须先按Order_id取Max(pay_amount)后再做SUM
- **口径**：预付费券（O2O券）专用口径
- **计算公式**：-
- **涉及字段**：pay_amount
- **样例SQL**：

        SUM(pay_amount)
FROM
        (
                SELECT 
                        order_id,MAX(pay_amount) AS pay_amount
                FROM
                        sbux_cdm.dwd_pmt_o2o_coupon_redemption_category_di
                WHERE
                        ds = '20260319'

## O2O Coupon Sold GMV
- **指标别名**：预付费券售卖金额
- **业务含义**：预付费券售卖金额
- **口径**：预付费券(O2O券）专用口径
- **计算公式**：-
- **涉及字段**：sold_amt
- **⚠️ 强制警告**：美团渠道的售卖金额必须单独从另一张表获取，必须使用 UNION ALL 合并两表数据，禁止只查询一张表
- **样例SQL**：
```sql
-- 非美团渠道
SELECT 
        sold_platform,
        SUM(CAST(sold_amt AS DOUBLE)) AS sold_amt
FROM 
        sbux_cdm_dev.dwd_pmt_o2o_coupon_sold_detail_di
WHERE 
        ds = '20260301'
GROUP BY 
        sold_platform

UNION ALL

-- 美团渠道（注意：该表字段与上述表不同，需对照数据字典确认字段）
SELECT 
        '美团券' AS sold_platform,
        SUM(sales_price) AS sold_amt 
FROM 
        sbux_cdm.dwd_pmt_meituan_sales_di
WHERE 
        ds = '20260301'
```

## Discount%
- **指标别名**：折扣率，全渠道折扣率，Discount Rate
- **业务含义**：折扣金额 / (销售额 + 折扣金额)
- **聚合方式**：分母：⚠️必须先按Order_id取Max(pay_amt)后再做SUM，分子：可直接SUM
- **口径**：OAH Discount%计算口径
- **计算公式**：-
- **涉及字段**：pay_amt,discount_amt
- **样例SQL**：
SELECT 
        SUM(discount_amt) / (SUM(pay_amt) + SUM(discount_amt))
FROM 
        (
                SELECT 
                        order_line_no
                        ,MAX(pay_amt) AS pay_amt 
                        ,SUM(discount_amt) AS discount_amt
                FROM 
                        sbux_cdm_dev.dwd_aop_mkt_order_line_discount_detail_di 
                WHERE 
                        ds >= '20260325' 
                GROUP BY 
                        order_line_no
        ) AS A
